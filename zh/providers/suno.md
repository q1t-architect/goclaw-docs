# Suno

> 使用 Suno 的 AI 音乐生成平台创作音乐和音频。

🚧 **本页面正在建设中。** 内容即将推出——欢迎贡献！

## 概述

Suno 是一个 AI 音乐生成 provider。GoClaw agent 可以使用 Suno 根据文本提示创作歌曲、生成背景音乐和制作音频片段。

## Provider 类型

```json
{
  "providers": {
    "suno": {
      "provider_type": "suno",
      "api_key": "your-suno-api-key"
    }
  }
}
```
