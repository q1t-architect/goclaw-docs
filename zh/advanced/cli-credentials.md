# CLI 凭据

> 安全存储和管理用于 shell 工具执行的命名凭据集。

🚧 **此页面正在建设中。** 内容即将推出。

## 概述

CLI 凭据让你可以定义命名凭据集（API key、token、连接字符串），agent 在通过 `exec` 工具运行 shell 命令时可以引用这些凭据 — 无需在系统提示词或对话历史中暴露密钥。内置常用工具的预设模板。
